pineapple:stop-test

Full name:

com.alpha.pineapple:pineapple-maven-plugin:1.0-SNAPSHOT:stop-test

Description:

Goal which executes a Pineapple stop-application-test operation on the Maven project which is defined by the input parameters.

If no parameters are supplied the goal will attempt to discover a project in the current runtime directory and execute the operation on the discovered project.

Attributes:

  • Requires a Maven 2.0 project to be executed.

Required Parameters

Name Type Since Description
buildDirectory File - This is where build results go.
environment String - Environment for target project.
finalName String - The Maven Project Object
session MavenSession - Maven session object.
settings Settings - Maven settings object.

Parameter Details

buildDirectory:

This is where build results go.
  • Type: java.io.File
  • Required: Yes
  • Expression: ${project.build.directory}

environment:

Environment for target project.
  • Type: java.lang.String
  • Required: Yes
  • Expression: ${pineapple.environment}

finalName:

The Maven Project Object
  • Type: java.lang.String
  • Required: Yes
  • Expression: ${project.build.finalName}

session:

Maven session object.
  • Type: org.apache.maven.execution.MavenSession
  • Required: Yes
  • Expression: ${session}

settings:

Maven settings object.
  • Type: org.apache.maven.settings.Settings
  • Required: Yes
  • Expression: ${settings}

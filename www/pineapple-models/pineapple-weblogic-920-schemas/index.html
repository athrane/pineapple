Overview

The WebLogic 9.2 schema project project defines a model for configuration for WebLogic servers. The projects implements a OXM mapping from the WebLogic 9.2.x XML schemas published by BEA to Java classes. The OXM mapping is implemented using XMLBeans.

More information about the WebLogic 9.2 schemas can be here.

Usage

Project schemas

The project uses multiple schemas for definition of models. The main domain schema defines the namespace:

  • http://www.bea.com/ns/weblogic/920/domain.

The domain schema uses three auxiliary security scehmas which defines the namespaces:

  • http://www.bea.com/ns/weblogic/90/security.
  • http://www.bea.com/ns/weblogic/90/security/wls.
  • http://www.bea.com/ns/weblogic/90/security/xacml.

Finally, the diagnostics schema defines the namespace:

  • http://www.bea.com/ns/weblogic/90/diagnostics.

For more information about where the schemas can be found, refer to the Schema locations for plugins page.

How-to: Install JRockit R27

Overview

This example illustrate another usage of the module defined in the Install JRockit R27 example to uninstall an existing JRockit R27 installation.

Invoke Pineapple to uninstall the JVM

Start your Pineapple client of choice:

  • Select the module where the installation is defined.
  • Select the environment for the model where the installation is defined.
  • Invoke the undeploy-configuration operation.

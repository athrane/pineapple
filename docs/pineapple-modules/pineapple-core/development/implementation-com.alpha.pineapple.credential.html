com.alpha.pineapple.credential description

Content

  • Credential definition
    • Contains classes used to or otherwise provide credentials to Pineapple resources. The credentials are used to access and authenticate the application with external resources.

Class diagram

Interfaces and classes

Credential

Defines interface for a security credential for a resource in an environment.

CredentialImpl

Implementation of the Credential interface.

CredentialProvider

Defines interface for credential providers which can supply Pinapple with security credentials when requested to.

XMLConfigurationCredentialProviderImpl

Implementation of the CredentialProvider interface which can initialize credentials from a Commons XMLConfiguration object.

FileCredentialProviderImpl

Implementation of the CredentialProvider interface which can initialize credentials from an XML file.

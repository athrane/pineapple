Create environment in the environment configuration

Steps

To create a new environment in the environment configuration:

  • Open the Configuration menu and select the Create Environment menu item. At the New Environment form enter:
  • At the ID field enter the name of the new environment.
  • At the Description field enter a description of the environment.

Result

The result after executing the steps are:

  • A new environment has been created in the environment configuration for resources and credentials.
  • If the environment already existed then the request was ignored.

Refresh Environment Configuration

Steps

To refresh the environment configuration:

  • Open the Configuration menu and select the Refresh Configuration menu item.

Result

The result after executing the steps are:

  • Refresh the environment configuration of Pineapple by:
    • Reloading the environment configuration files which defines the known set of environments, resources and credentials.
    • Reloading the known set of modules.
  • A report is created which details how the initialization went. The report will be available in the reports view.

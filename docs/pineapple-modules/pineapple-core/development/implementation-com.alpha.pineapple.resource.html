Interfaces and classes

ResourceFactory

Interface for abstract factories which are used to create the instances of the object which implements a plugin.

The factory creates these kind of objects implemented by a plugin:

  • Session � establishment of connection to external resources
  • Visitor � implement traversals of configuration model

Configuration

To use the OXM classes include a Maven dependency to this project:

  <dependency>
    <groupId>com.alpha.pineapple</groupId>
    <artifactId>pineapple-weblogic-jmx-920-schemas</artifactId>
        <version>1.0.0</version>
  </dependency>

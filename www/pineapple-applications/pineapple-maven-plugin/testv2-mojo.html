pineapple:testv2

Full name:

com.alpha.pineapple:pineapple-maven-plugin:1.0-SNAPSHOT:testv2

Description:

Executes pineapple test operation.

Attributes:

  • Requires a Maven 2.0 project to be executed.

Required Parameters

Name Type Since Description
artifactId String - ArtifactId for target project.
environment String - Environment for target project.
groupId String - GroupId for target project.
session MavenSession - Maven session object.
settings Settings - Maven settings object.

Parameter Details

artifactId:

ArtifactId for target project.
  • Type: java.lang.String
  • Required: Yes
  • Expression: ${pineapple.artifactId}

environment:

Environment for target project.
  • Type: java.lang.String
  • Required: Yes
  • Expression: ${pineapple.environment}

groupId:

GroupId for target project.
  • Type: java.lang.String
  • Required: Yes
  • Expression: ${pineapple.groupId}

session:

Maven session object.
  • Type: org.apache.maven.execution.MavenSession
  • Required: Yes
  • Expression: ${session}

settings:

Maven settings object.
  • Type: org.apache.maven.settings.Settings
  • Required: Yes
  • Expression: ${settings}

Plugin Documentation

Goals available for this plugin:

GoalDescription
pineapple:install-appGoal which executes a Pineapple install-application operation on the Maven project which is defined by the input parameters.

If no parameters are supplied the goal will attempt to discover a project in the current runtime directory and execute the operation on the discovered project.

System Requirements

The following specifies the minimum requirements to run this Maven plugin:

Maven2.0
JDK1.5
MemoryNo minimum requirement.
Disk SpaceNo minimum requirement.

Usage

You should specify the version in your project's plugin configuration:

<project>
  ...
  <build>
    <!-- To define the plugin version in your parent POM -->
    <pluginManagement>
      <plugins>
        <plugin>
          <groupId>com.alpha.pineapple</groupId>
          <artifactId>pineapple-maven-plugin</artifactId>
          <version>1.0-SNAPSHOT</version>
        </plugin>
        ...
      </plugins>
    </pluginManagement>
    <!-- To use the plugin goals in your POM or parent POM -->
    <plugins>
      <plugin>
        <groupId>com.alpha.pineapple</groupId>
        <artifactId>pineapple-maven-plugin</artifactId>
        <version>1.0-SNAPSHOT</version>
      </plugin>
      ...
    </plugins>
  </build>
  ...
</project>

For more information, see "Guide to Configuring Plug-ins"

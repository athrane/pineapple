Overview

The JRockit installation plugin implements functionality to install the JRockit JVM and validate the installation. For more information about the JRockit JVM, please refer to the page The official Oracle JRockit JVM home page.

The plugin supports the installation of these releases of JRockit:

  • R27
  • R28

Usage

Go to the Plugin Usage page for instructions on using the plugin.

Supported operations

The plugin supports the default operations:

  • deploy-configuration (from the deployment workflow). When the plugin is invoked with this operation, the plugin installs the JRockit JVM.
  • undeploy-configuration (from the deployment workflow). When the plugin is invoked with this operation, the plugin uninstalls the JRockit JVM.
  • test (from the test workflow). When the plugin is invoked with this operation, the plugin validates the whether the specified JVM is installed.

For more information about the default operations, refer to the Operation and workflow reference.

Plugin schema

The plugin defines a schema named JRockit installation plugin schema for definition of models in module model files. The schema defines the namespace http://pineapple.dev.java.net/ns/plugin/jrockit_installation_1_0. For more information about where the schema can be found, refer to the Schema locations for plugins page.
